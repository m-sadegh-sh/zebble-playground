*************************************************
* PREPARE YOUR VISUAL STUDIO « Once is enough » *
*************************************************
1: ⇒ Tools → Options → [NuGet] Package Manager → General
        ⚠ Make sure your Visual Studio can restore NuGet packages

2: ⇒ Tools → Options → Text Editor → File Extension
        ⚠ Add an entry for "zbl" file extension to be edited using the XML editor.

3: ⇒ Debug → Windows → Exception Settings
        ⚠ Click "Common Language Runtime Exception" to show a tick icon
